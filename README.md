📌 Simple Console To-Do App (JavaScript)

📖 About The Project
This is a simple console-based To-Do application built using JavaScript.
The user interacts with the app using prompt() and console.log() in the browser console.

It allows users to:

Add tasks
List all tasks
Delete tasks
Quit the application

🚀 Features
Add new tasks
View all tasks with index numbers
Delete tasks using index
Continuous loop until user enters "quit"

🛠️ Technologies Used
HTML
JavaScript (Vanilla JS)
Browser Console
prompt() and console.log()

📂 Project Structure
📁 Todo-App
 ├── index.html
 ├── app.js
 └── README.md
 
▶️ How To Run
Open index.html in your browser.
Open the browser console (Right click → Inspect → Console).
Follow the instructions:
Type "add" to add a task
Type "list" to view tasks
Type "delete" to remove a task
Type "quit" to exit
